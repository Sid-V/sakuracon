# Sakura-Con 2026 Schedule

A mobile-first web app for browsing the [Sakura-Con 2026](https://sakuracon.org) event schedule — April 3–5, Seattle Convention Center.

**Live site:** [sakuracon.vercel.app](https://sakuracon.vercel.app)

## Features

- Browse all panels, workshops, screenings, and events
- Filter by day, venue, age rating, and tags
- Search events by title, location, or description
- Save favorites — your personalized schedule persists in the browser
- Venue maps with direct links to interactive floor plans
- Dark mode support
- Auto-hiding filter bar for maximum screen space on mobile

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com)
- **Icons:** [Lucide React](https://lucide.dev)
- **Hosting:** [Vercel](https://vercel.com)
